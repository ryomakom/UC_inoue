旧統一教会が参院選に与えた影響の推定

朝日新聞デジタル企画報道部　小宮山亮磨 @ryomakom

2023-05-21

この文書について

　2023年度の日本選挙学会総会でのポスター発表「旧統一教会が参院選に与えた影響の推定」の分析手法や結果について、もとになったデータやRのコードとともに公開する。詳細はポスターに譲るが、自民党の井上義行氏（安倍晋三元首相の元秘書官）の得票を分析することで、旧統一教会が参院選に与えた影響を定量的に見積もろうとしている。

必要なパッケージをとってくる

library(tidyverse)
library(modelsummary)
library(MatchIt)
library(rgenoud)
library(cobalt)

元となるデータをとってきて加工する

　参院選の得票を記録した朝日新聞の社内データベースと、旧統一教会がウェブサイトで公表している「家庭教会」という教団施設の住所を元に作ったCSVファイルを読み込む。教団施設の有無は「UC有無」とタグ付けしている。「HS有無」「HC有無」はそれぞれ「幸福の科学」の施設と保健所の有無を意味しているが、今回の分析には使っていない。その他のデータは政府の「社会・人口統計体系」をeStatから取得して加工したもの。

inoue <- read_csv("data/UC_inoue.csv") %>%
  mutate(inoue_ratio=井上得票*100/投票者数, # 井上氏の得票率と
         ldp_ratio=(自民得票計-井上得票)*100/投票者数) %>% # 井上氏を除いた自民の得票率を算出
  dplyr::select(-井上得票,-自民得票計,-投票者数) %>%
  pivot_wider(names_from = 年,
              values_from = c(inoue_ratio,
                              ldp_ratio)) %>%
  mutate(inoue_ratio_change=inoue_ratio_2022-inoue_ratio_2019, # 井上氏の得票率変化量と
         ldp_ratio_change=ldp_ratio_2022-ldp_ratio_2019) %>% # 自民得票率の変化量を算出
  dplyr::select(-inoue_ratio_2022,
                -inoue_ratio_2019,
                -ldp_ratio_2022,
                -ldp_ratio_2019) %>% 
  dplyr::select(都道府県市区町村,
                inoue_ratio_change,
                UC有無,
                HS有無,
                HC有無,
                ldp_ratio_change,
                `A2301_住民基本台帳人口（総数）`,
                everything()) %>% #列の順番並べ替え
  select_if(negate(anyNA)) #NAのある列を除外

井上氏の得票に各変数が与える効果を調べる

　各変数を標準化したうえで、井上氏の得票率変化量を目的変数とした単回帰分析をして、それぞれの効果量を調べる。

# 自民党得票率とUC有無、HS有無、HC有無以外のすべての説明変数を人口で割る
for(i in 8:73){
  inoue[,i] <- inoue[,i]/inoue[,7]
}

# 井上氏の得票率変化量とUC有無、HS有無、HC有無以外のすべての説明変数を標準化
for(i in 6:73){
  inoue[,i] <- inoue[,i] %>%
    scale()
}

# 分析のためにデータフレーム形式に変える
inoue <- inoue %>% 
  as.data.frame()


# 各変数と井上氏の得票率変化量との単回帰分析をする
resFrame1<-data.frame() # forループのたびに上書きする一時的なデータフレーム
resFrame2<-data.frame() # 最終結果を書き込むデータフレーム

for(i in 6:73){
  result <- summary(lm(inoue$inoue_ratio_change~inoue[,i]))
  
  coef <- result$coefficients
  
  resFrame1 <- coef # i番目の出力結果を代入
  resFrame2 <- rbind(resFrame2,resFrame1) # resFrame2の末行にresFrame1を代入
}

マッチングをしていく

　67項目ある変数全部を考慮してマッチングするのがモデルA、67項目のうち効果量の大きい9項目にしぼったものがモデルB、8分野のうちそれぞれ効果量が最大の8項目にしぼったものがモデルCとする。いずれのモデルでも、これらのほかに人口と、井上氏を除いた自民党全体の得票率はマッチングの要素に組み込む。A、B、Cそれぞれのモデルについて、「遺伝的マッチング」と「最近傍マッチング」を適用する。
